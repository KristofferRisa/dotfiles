# Explain Code

## Purpose
Provide clear, comprehensive explanations of code for learning and documentation.

## Instructions
Analyze the provided code and explain:

1. **High-Level Overview**
   - What the code does (purpose)
   - Main inputs and outputs
   - Key components

2. **Step-by-Step Breakdown**
   - Walk through the logic flow
   - Explain each significant section
   - Clarify any complex operations

3. **Key Concepts**
   - Programming patterns used
   - Algorithms or data structures
   - Language-specific features

4. **Dependencies & Context**
   - Required libraries or frameworks
   - Assumptions made
   - External resources used

5. **Potential Issues or Considerations**
   - Edge cases
   - Performance implications
   - Security considerations

## Style
- Use analogies for complex concepts
- Assume the reader knows the programming language basics
- Highlight non-obvious or clever implementations
- Be encouraging and educational
- Use comments inline where helpful

## Output Format
```markdown
## Overview
[Brief description]

## Code Walkthrough
### Section 1: [Name]
[Explanation]

### Section 2: [Name]
[Explanation]

## Key Concepts
- **[Concept]**: [Explanation]

## Notes
- [Important consideration]
```
